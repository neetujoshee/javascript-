JS Camel Case

1. Underscore: first_name, last_name
2. Upper Camel Case: FirstName, GetElementByID
3. Lower Camel Case: Starts with a lowercase letter: javaScript, firstName

Variables in JavaScript

1. Automatically (implicit declaration)
2. Using `var`; if you support old browsers
3. Using `let`; if you need block-scoped variables
4. Using `const`; if the value should not be changed
