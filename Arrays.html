Array

1. It is an object that represents a collection of
similar type of elements.
2. It is a special variable that holds more than one value at
a time.
3.there are three ways to construct an array
i. Conventional array
ii. Condensed Array
iii. Literal array

Conventional Arrays

syntax: var name=newarray();
it creates an empty array.

2. Condensed Array

It creates an array with some length
but the slots are empty.

3. Literal Array

Common way to create an array.
